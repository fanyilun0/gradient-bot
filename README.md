## usage
1. `git clone https://github.com/fanyilun0/gradient-bot.git`
2. `cd gradient-bot`
3. `npm install -g pm2 && npm install`
4. modify file `ecosystem.config.js` with your own account and proxy
5. `pm2 start ecosystem.config.js`
